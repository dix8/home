01. 关于我

你好！我是 Alex Chen，一名充满热情的全栈开发者和 UI 设计师。 我专注于构建美观、实用且高性能的 Web 应用程序。

从学生时代开始接触编程，到现在已经有多年的开发经验。 我相信好的设计不仅仅是视觉上的美感，更是用户体验的完美结合。 每一个像素、每一行代码都应该有其存在的意义。

在工作之余，我喜欢探索新技术、开源贡献、摄影和旅行。 我始终保持学习的热情，追求卓越的技术和设计。
